# Bot de Discord - Concurso Anual de Juegos

Bot para llevar el registro y ranking del concurso anual de juegos completados entre amigos.

## Características

- Sistema de registro de juegos con aprobación de admins
- Ranking en tiempo real
- Estadísticas detalladas por usuario y grupo
- Dashboard visual completo
- Re-completados permitidos con puntos completos
- Regla especial de Elkie para premios

## Comandos Principales

- `/registrar` - Registrar un juego completado
- `/ranking` - Ver el ranking general
- `/tablero` - Dashboard completo con estadísticas
- `/mi-posicion` - Ver tu posición actual
- `/reglas` - Ver las reglas del concurso

## Tecnologías

- Python 3.12
- discord.py 2.4.0
- SQLite (aiosqlite)
- Railway (hosting)
